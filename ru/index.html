ПЕРСОНАЛЬНАЯ ИНФОРМАЦИЯ
Персональная информация — это данные, которые могут быть использованы для уникальной идентификации или связи с одним человеком. Мы НЕ СОБИРАЕМ, НЕ ХРАНИМ и НЕ ИСПОЛЬЗУЕМ какую-либо персональную информацию.
НЕПЕРСОНАЛЬНАЯ ИНФОРМАЦИЯ
Неперсональная информация — это данные в форме, которая не допускает прямой ассоциации с каким-либо конкретным лицом. МЫ ИСПОЛЬЗУЕМ неперсональную информацию, такую ​​как объем памяти вашего устройства, модель телефона, свойства экрана, только для информирования пользователей.
РАЗРЕШЕНИЯ
Примечание: Ни одно из ниже перечисленных разрешений выдается без вашего ведома. Все разрешения запрашиваются динамически через диалоговые окна.
Для реализации основных функций приложения требуются разрешения на доступ к локальному медиаконтенту.
READ_EXTERNAL_STORAGE: Позволяет приложению считывать данные с внешнего хранилища. (Android 12 (api 32) и ниже)
WRITE_EXTERNAL_STORAGE: Позволяет приложению записывать (также означает: переименовывать/копировать/удалять) данные на внешнее хранилище (кроме SD-карт). (Android 10 (api 29) и ниже)
READ_MEDIA_VIDEO/READ_MEDIA_AUDIO: Позволяет приложению считывать видео/аудиофайлы с внешнего хранилища. (Android 13 (api 33) и выше)
MANAGE_EXTERNAL_STORAGE: Позволяет приложению записывать (также означает: переименовывать/копировать/удалять) данные на внешнее хранилище/SD-карту. Данное разрешение может быть отозвано в любое время. (Android 11 (api 30) and above)
Дополнительно
RECORD_AUDIO: Для работы Аудио-Визуализатора требуется это разрешение. Ничего не записывается, разрешение используется только для аудиовизуализации и может быть отозвано в любое время.
WRITE_SETTINGS: Это разрешение требуется для изменения яркости из приложения. Разрешение можно отозвать в любое время.
Примечание: Внешним хранилищем может быть съемное хранилище (например, SD-карта) или внутреннее хранилище вашего устройства (несъемное).
Примечание: Разрешения применяются только к аудио- и видеофайлам, все остальные типы файлов скрыты от приложения.
