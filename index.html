Non-personal information is data in a form that does not allow direct association with any specific person. We USE non-personal information, such as your device memory size, phone model, screen properties, only to inform users.
PERMISSIONS
To implement the basic functions of the application, it requires permissions to access local media content.
READ_EXTERNAL_STORAGE: Allows an application to read from external storage. (Android 12 (api 32) and below)
WRITE_EXTERNAL_STORAGE: Allows an application to write (also means: rename/copy/delete) to external storage (except SD cards). (Android 10 (api 29) and below)
READ_MEDIA_VIDEO/READ_MEDIA_AUDIO: Allows an application to read video/audio files from external storage. (Android 13 (api 33) and above)
MANAGE_EXTERNAL_STORAGE: Allows an application to write (also means: rename/copy/delete) to external storage / SD cards. (Android 11 (api 30) and above)
Note: External storage can be removable storage (such as an SD card) or your device's internal storage (non-removable).
